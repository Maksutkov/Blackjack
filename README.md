# Blackjack
Little C++ project. Console Blackjack game


# Rules:
1. Each player starts with two cards. Players see only the dealer's first card;
   they can draw one card each turn. The game ends when everyone has passed.
2. The player who reaches 21 wins.
3. If no one reaches 21, the player with the highest score, but not exceeding 21, wins.
4. If someone reaches more than 21, they automatically lose;
   if everyone reaches more than 21, the player with the lowest score wins.
5. If the dealer and player have the same number of points, the game is declared a draw.

# Правила: 
1. Со старта у каждого по 2 карты. Игрок видит только первую карту дилера;
   в каждый ход можно взять одну карту. Игра подойдёт к концу, когда все спасовали
3. Побеждает человек, который набирает 21.
4. Если никто не набрал 21, то побеждает тот, кто набрал больше, но не привысил 21.
   В случае набирания больше чем 21 происходит автоматический проигрыш.
   Если больше 21 набрали все, то побеждает тот, кто набрал меньше.
5. В случае одинакового количества очков у дилера и игрока объявляется ничья.
